This program currently requires a few step from the user

Canvas

1. Go into Canvas on a desktop
2. Go into settings
3. Scroll down to "Approved integrations"
4. Click new access token
5. Save and store the token they provide
    
Notion

1. Set up account if haven't already
2. Create a new page
3. Share the page with the public Integration
4. Copy the page ID
